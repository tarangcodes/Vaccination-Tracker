# Kiddo Vaccinations

A simple, client-side web application to help parents track childhood vaccinations for children from birth to 11 years. This project is built with plain HTML, CSS, and JavaScript, using browser localStorage for data persistence.

## ‚ú® Features

- **User Accounts**: Simple registration and login system.
- **Child Profiles**: Add and manage multiple child profiles.
- **Vaccine Tracking**: View a detailed list of recommended vaccines, mark them as "Done" or "Pending", and track progress.
- **Dashboard**: A central hub to view all children and their overall vaccination progress.
- **Printable Certificate**: Generate and print a simple vaccination certificate for any child.
- **Parent Directory**: Keep a record of parent/guardian details.

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
